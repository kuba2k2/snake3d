Sterowanie:
Wężem sterujemy używając strzałek, klawiszy WASD lub myszką.
Klawiszem F5 zmieniamy położenie (tryb) kamery.
Aby zakończyć grę naciskamy Esc, aby zastopować naciskamy klawisz Enter.
